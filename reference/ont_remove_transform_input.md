# Remove Transform Input

Removes an input dataset from a transform.

## Usage

``` r
ont_remove_transform_input(transform_id, input_dataset_id, con = NULL)
```

## Arguments

- transform_id:

  The transform identifier.

- input_dataset_id:

  The input dataset ID to remove.

- con:

  Optional DBI connection.

## Value

Invisibly returns TRUE.
