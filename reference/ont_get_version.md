# Get Concept Version

Get Concept Version

## Usage

``` r
ont_get_version(concept_id, scope, version, con = NULL)
```

## Arguments

- concept_id:

  Character. The concept ID.

- scope:

  Character. The scope.

- version:

  Integer. The version number.

- con:

  A DBI connection.

## Value

A single-row tibble with version details.
