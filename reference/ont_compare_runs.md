# Compare Runs

Compares two runs, showing differences in inputs, outputs, and
execution.

## Usage

``` r
ont_compare_runs(run_id_1, run_id_2, con = NULL)
```

## Arguments

- run_id_1:

  First run ID.

- run_id_2:

  Second run ID.

- con:

  Optional DBI connection.

## Value

A list with comparison details.
